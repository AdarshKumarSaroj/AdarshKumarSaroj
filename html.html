HTML Basics

This is an HTML project that covers various HTML elements:

  • Head: Contains metadata about the document.
  • Body: Contains the visible content of the document.
  • Title: Sets the title of the document.
  • Header: Contains introductory content.
  • Heading Tags: Used for headings (

    to

    ).
  • Paragraph: Represents a paragraph of text.
  • Line Break: Creates a line break within text.
  • Bold: Makes text bold.
  • Italic: Makes text italic.
  • Underline: Underlines text.
  • Style: Defines internal CSS styles.
  • Horizontal Rule: Creates a horizontal line.
  • Font Size Attribute: Sets the font size.
  • Color Attribute: Sets text color.
  • Unordered List: Creates a bulleted list.
  • Ordered List: Creates a numbered list.
